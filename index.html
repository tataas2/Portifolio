Sobre Mim

Sou estudante de Ciência da Computação e formada em Técnico em Análise e Desenvolvimento de Sistemas, com foco em desenvolvimento back-end. Tenho me aprofundado em C#, ASP.NET Core, Entity Framework e bancos de dados como SQL Server e PostgreSQL. Estou aprendendo a construir APIs RESTful seguindo boas práticas de arquitetura e segurança. Embora meu foco principal seja o back-end, também estudo front-end com HTML, CSS, Bootstrap e React, visando compreender o desenvolvimento web de forma completa. Tenho interesse em cibersegurança e inteligência artificial, áreas que venho explorando com dedicação para integrar em projetos futuros.

Meus Projetos

Pizzaria

Projeto temático desenvolvido com HTML, CSS e JavaScript, focado em front-end. A aplicação simula o site de uma pizzaria inspirada no universo Marvel, com layout responsivo, interações dinâmicas e identidade visual personalizada. Idealizado para exercitar estruturação de páginas, manipulação do DOM e estilização criativa.
